# Product Dashboard

## Project Description
The Product Dashboard is a React application that displays a list of products with their prices and stock status. Users can view products, identify which items are out of stock through special styling, and remove products from the dashboard by clicking the remove button.

## Features
- Display product cards
- Show product name, price, and stock status
- Highlight out-of-stock products with conditional styling
- Remove products dynamically
- Responsive and styled dashboard layout
